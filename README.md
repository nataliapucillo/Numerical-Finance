# Numerical Finance Notebooks

This repository collects a set of notebooks on Monte Carlo methods and option pricing, based on topics covered in my *Numerical Finance* course.

## Contents

- **Asian Option Pricing**  
  Monte Carlo pricing of Asian options under the Black–Scholes framework.

- **Variance Reduction Techniques**  
  Standard Monte Carlo, antithetic variables, control variates, and importance sampling.

- **Discretization Schemes for Monte Carlo Methods**  
  Comparison of Euler and Milstein discretization.

- **Longstaff–Schwarz Method for American Option Pricing**  
  Monte Carlo pricing of American options with least-squares regression.
